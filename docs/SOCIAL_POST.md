More Technical Post — LinkedIn

For the Breaking Apps Hackathon, I built a small but complete Passmark + Playwright regression suite for my deployed Lysergic Systems web app.
The objective was not to overbuild. It was to establish a practical first regression layer for a real public app.
The suite validates:
1. Homepage smoke behavior
2. Primary navigation
3. Mobile viewport usability
Final run:
3 passed
0 failed
What made this useful:
- Passmark kept the test intent readable through plain-English user flows.
- Playwright handled real browser execution.
- OpenRouter powered the Passmark AI gateway.
- GitHub Codespaces provided a fast reproducible environment.
